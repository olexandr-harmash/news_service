## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```


## Running the app docker-compose

```bash
# build containers
$ docker-compose build
# up containers with watch mode
$ docker-compose up
```
